# 创建自定义过滤器

> 创建自定义过滤器源码示例


## 开发栈


- Vue 2.0
- ES6
- Browserify
- Karma
- Mocha

## 安装使用

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# lint all *.js and *.vue files
npm run lint

# run unit tests
npm test
```

For more information see the [docs for vueify](https://github.com/vuejs/vueify).
